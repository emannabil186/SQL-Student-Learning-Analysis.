# Student Learning Environment Analysis (SQL Case Study)

## 📌 Project Overview
This project focuses on analyzing data related to the student learning environment, assessing how factors like internet quality, financial status, and lecture durations impact students' adaptability and academic performance. 

The analysis was conducted using advanced SQL queries to extract meaningful insights that can help educational institutions optimize their digital learning frameworks.

## 📊 SQL Skills Demonstrated
The project includes **10 comprehensive SQL queries** covering essential data analysis techniques:
* **Data Aggregation & Statistics:** Calculating averages, counts, and percentages for student groups.
* **Multi-Table Joins:** Combining relevant datasets to find correlations.
* **Conditional Logic (CASE Statements):** Segmenting students based on specific environment thresholds.
* **Subqueries & CTEs:** Isolating specific metrics for deeper analytical layers.
* **Database Views:** Creating reusable virtual tables for simplified reporting.

## 📁 Project Documentation & Files
* `code.sql` - Full clean SQL script containing the 10 analytical queries and query logic.
* `sql_finale_project (3).html` - Interactive export of query results and formatted tables.
* `sql_finale_project.pdf` - Comprehensive analytical report summary and presentation overview.

## 💡 Key Analytical Insights
* Identified student segments most vulnerable to poor network infrastructures.
* Calculated standard learning metrics and average lecture engagement rates across different demographics.
